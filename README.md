Overview
This project presents an interactive COVID-19 dashboard built using Microsoft Power BI. It transforms raw COVID-19 data into meaningful insights through data cleaning, modeling, and visualization. The dashboard enables users to monitor global pandemic trends and compare statistics across countries.
🎯 Objectives
Analyze global COVID-19 trends.
Visualize confirmed, recovered, active, and death cases.
Compare country-wise statistics.
Demonstrate Power BI data visualization and reporting skills.
🛠️ Tools & Technologies
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV
📊 Dashboard Features
KPI cards for Total Cases, Recoveries, Deaths, and Active Cases
Country-wise comparison
Time-series trend analysis
Interactive slicers and filters
Dynamic charts and visualizations
📈 Skills Demonstrated
Data Cleaning & Transformation
Data Modeling
DAX Calculations
Data Visualization
Dashboard Design
Business Intelligence Reporting
